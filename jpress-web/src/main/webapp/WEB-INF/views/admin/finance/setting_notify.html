收款通知

请先在 [常规设置] 开填写管理员邮箱。

请先在 [常规] 里填写管理员邮箱，并在 [通信] 开启邮件功能后才能使用。

请先在 [常规设置] 开填写管理员的手机号。

请先在 [通信] 开启短信功能，这里填写阿里云或腾讯云的短信模板ID。

请填写阿里云或腾讯云的短信签名。

请先在 [常规] 里填写管理员的微信 OpenId，并在 [微信基础设置] 完成微信配置才能正常使用。
